Hari ini saya mempelajari cara membuat tampilan halaman Profile Page yang berisi data diri, dengan memanfaatkan framework CSS Bootstrap. Saya mulai dengan memahami bagaimana cara menggunakan Bootstrap melalui CDN (Content Delivery Network). Metode ini sangat praktis karena memungkinkan saya langsung mengakses semua fitur dan komponen Bootstrap tanpa perlu menginstal atau mengunduh file ke lokal. Pendekatan ini sangat cocok untuk latihan ataupun proyek-proyek skala kecil, terutama saat sedang belajar.

Fokus pembelajaran saya selanjutnya adalah pada komponen Navbar, yang berfungsi sebagai menu navigasi utama dalam sebuah halaman web. Saya belajar bahwa Navbar sangat fleksibel dan dirancang untuk responsif—dapat menyesuaikan tampilannya secara otomatis berdasarkan ukuran layar. Misalnya, ketika ditampilkan di perangkat dengan layar besar, navigasi akan muncul secara penuh. Namun jika dilihat di perangkat berlayar kecil, akan muncul tombol hamburger yang bisa diklik untuk menampilkan menu.

Selama proses belajar, saya mulai terbiasa untuk membaca dan menelusuri dokumentasi resmi Bootstrap di situs getbootstrap.com. Meskipun awalnya cukup membingungkan karena banyak istilah teknis, seiring waktu saya merasa lebih terbiasa. Kuncinya adalah membaca perlahan, memahami contoh kode, lalu langsung mencobanya di browser. Dengan begitu, saya bisa langsung melihat bagaimana perubahan kode memengaruhi tampilan halaman. Hari ini jadi salah satu pembelajaran yang menyenangkan karena tidak hanya menambah pengetahuan teknis, tapi juga membuat saya lebih percaya diri dalam membangun antarmuka web yang rapi dan responsif.
