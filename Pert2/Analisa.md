Hari ini saya mempelajari materi seputar Bootstrap melalui dokumentasi resmi di laman getbootstrap.com. Fokus utama pembelajaran adalah pada bagian CDN Links dan komponen Navbar. Saya mulai dengan memahami bagaimana cara mengintegrasikan Bootstrap ke dalam halaman HTML. Bootstrap dapat digunakan dengan dua metode utama, yaitu dengan mengunduh dan menyimpan file CSS serta JavaScript secara lokal, atau dengan cara yang lebih praktis menggunakan Content Delivery Network (CDN). Saya memilih metode CDN karena lebih efisien dan tidak memerlukan pengunduhan file. Integrasi ini cukup dilakukan dengan menambahkan tag <link> dan <script> di dalam elemen <head> HTML. Dengan begitu, seluruh fitur dan komponen Bootstrap dapat langsung digunakan tanpa konfigurasi tambahan.

Selanjutnya, saya mempelajari salah satu komponen penting yaitu Navbar. Komponen ini sangat sering digunakan karena berfungsi sebagai menu navigasi utama dalam sebuah website. Dari dokumentasi, saya melihat struktur dasar Navbar yang menggunakan class navbar, navbar-expand-lg, dan beberapa class lainnya yang mendukung tampilan responsif. Saya memahami bahwa class navbar-expand-lg digunakan agar menu navigasi terlihat penuh di layar besar, sedangkan navbar-toggler berfungsi sebagai tombol hamburger pada tampilan layar kecil. Ada juga class tambahan seperti bg-light, navbar-dark, dan bg-dark yang dapat digunakan untuk mengatur tampilan warna background dan teks sesuai kebutuhan desain.

Dari pembelajaran ini, saya menyimpulkan bahwa penggunaan CDN untuk Bootstrap sangat membantu dalam mempercepat proses pengembangan, terutama dalam tahap awal pembuatan halaman web. Di sisi lain, komponen Navbar menunjukkan bagaimana Bootstrap mendukung konsep desain yang responsif dan fleksibel hanya dengan menambahkan beberapa class. Saya merasa dokumentasi Bootstrap sangat mudah dipahami karena setiap bagian dilengkapi dengan contoh kode dan penjelasan yang dapat langsung diterapkan dalam praktik. Pembelajaran hari ini sangat bermanfaat dalam membangun pemahaman dasar tentang struktur dan implementasi komponen dalam Bootstrap.
